

## Overview

**SuperSMS**, 这是一个短信群发的Android应用，可以群发短信推广。

## How to use


###### 1、新建一个文本文档，命名为number.txt,然后每行添加一个手机号码
###### 2、新建一个文本文档，命名为content.txt,添加你要发送的短信内容. 
###### 3、将number.txt和content.txt上传到SD卡的根目录，然后运行程序，点击Start开始群发。
